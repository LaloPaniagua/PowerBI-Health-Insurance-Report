This little proyect is a simple dashboard that breaks down health insurance costs using parameters such as age, gender, location (region), BMI and whether or not a patient is a smoker.

The main goal is to utilize a wide variety of visualization options in order to understand the best use cases for each one of them, this knowledge will be very usefull when creating more complex dashboards, and should make it easier to decide which one is more suited to the message I want to comunicate to stakeholders.

The dataset that I used is the "Medical Cost Personal Dataset" by PAVAN KUMAR, this dataset can be found on Kaggle at https://www.kaggle.com/code/mragpavank/medical-cost-personal-datasets

Making this proyect allowed me to test my skills in PowerBI, since it required me to clean the dataset, create new columns that allowed for easier categorization, and 
finally, create new visuals that may aid in the decision making process by making it easier to point out trends amongst different groups.
